# WebApp OWASP Top 10 Lab Guide

## Lab Setup
- Use intentionally vulnerable app in isolated local VM/container.
- Never target production or unauthorized systems.

## Exercise Plan
1. Broken Access Control: bypass role checks and identify missing authorization.
2. Injection: test unsafe query input and apply parameterized fixes.
3. Security Misconfiguration: audit headers and default credentials.
4. Vulnerable Components: identify outdated libraries and patch path.

## Deliverable
- For each issue: exploit steps, impact, fix, and verification test.
